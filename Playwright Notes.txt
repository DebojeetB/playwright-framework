4. There are different methods to handle network calls

    i. Route.fulfill --> In this method, a mock response can be passed in browser through fulfill method to validate results on UI
    ii. Route.continue --> In this method, request can be tweaked to check if application works fine. E.g. if a site's endpoint is updated then ideally it should through forbidden error. This we can check through Continue.
    iii. Route.abort --> This method is used if we want to block some API calls like if we want to block the images shown in UI or if we want to block the CSS then this method should be used.

7. To execute the test case in specific mode, like on multiple browsers/capture video/screenshots/logs- need to make changes in playwright.config.js file. 

use:{
browserName: 'chromium', --> Here any browser name can be added
headless: false,
viewport: {height:720, width:720},  --> this is particularly to test the flow in mentioned window size
video: 'retain-on-failure',
trace: 'on'
}

8. retires: 1 --> Use it to rerun a test case if it has failed. Add it inside package file under config.

9. workers: 3 --> Use it to execute tests in parallel. If 3 is declared, then 3 different test classes would be executed at once in parallel. Maximum 5 tests can be executed by default by Playwright. Add it inside package file under config. 
Note- it can only be applied on test files and not on test cases added inside a single class file.

10. To execute multiple test cases added in a single file parallelly, add the below code in class file after declaring test
 test.describe.configure({mode:'parallel'});

Note- Mode can be kept to serial- in this case test cases are interdependent. If first test case fails, reset all will be skipped.

12. Allure is a report view tool through which execution report of test cases can be viewed. To install Allure, type below in Playwright
npm i -D @playwright/test allure-playwright

===== > To run the test case and view report through Allure type below commands-
npx playwright test tests/Transaction_TC1.spec.js --reporter=line,allure-playwright
 
==== > Once test case has passed, type below  (Generate report)
npx allure generate ./allure-results --output ./allure-report 

==== > Now open report
npx allure open ./allure-report

14. To handle multiple tabs, use below code

const [pageTwo]= await Promise.all([
        this.page.context().waitForEvent('page'),
        await this.page.locator(".blinkingText").click()
    ])
    await pageTwo.waitForLoadState();

Now again to redirect to parent tab, use
await this.page.bringToFront();

To redirect again to the child tab, use
await pageTwo.bringToFront();
